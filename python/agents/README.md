# Agents Overview
This directory contains individual AI agent samples built using the Google Agent Development Kit (ADK).

## Categorization
| Agent | Description | Complexity |
|-------|-------------|------------|
| `time_teller` | Simple tool-using agent that tells the current time. | Basic |

## Development
To add a new agent:
1. Create a new directory under `python/agents/`.
2. Add an `agent.py` and a `README.md`.
3. Test it using `make run-agent NAME=your_agent_name`.
